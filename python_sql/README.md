## Running Quick start

Create a virtual environment and install prerequisites:

```ps1
# make sure your current directory is python_sql
python3 -m venv .sql
.sql\scripts\activate.ps1
pip install --upgrade pip
pip install -r requirements.txt
```

Run:

```ps1
python main.py
```
